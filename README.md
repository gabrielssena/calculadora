# Projeto Calculadora em React Expo
Para esse projeto vamos ter os seguintes Requisitos Funcionais e não funcionais:

Requisitos Funcionais: 
RF-01: A aplicação deve permitir o usuário digitar números de 0 a 9;
RF-02: A aplicação deve permitir o usuário a escolher a operação matemática (Adição, Divisão, Multiplicação);
RF-03: A aplicação deve mostrar o resultado da conta em tempo real;
RF-04: A aplicação deve permitir números do tipo float;
RF-05: A aplicação deve calcular raiz quadrada;
