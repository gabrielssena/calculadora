# Projeto Calculadora em React Expo
Para esse projeto vamos ter os seguintes Requisitos:
<br>
Requisitos:<br>
R-01: A aplicação deve permitir o usuário digitar números de 0 a 9; <br>
R-02: A aplicação deve permitir o usuário a escolher a operação matemática (Adição, Divisão, Multiplicação); <br>
R-03: A aplicação deve mostrar o resultado da conta em tempo real; <br>
R-04: A aplicação deve permitir números do tipo float; <br>
R-05: A aplicação deve calcular raiz quadrada; <br>
R-06: A aplicação deve permitir alternar o sinal do número (+/−); <br>
